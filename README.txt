Commands needed to run Poject:

API:
- pip install Flask     
- pip install fastapi  
- pip install flask-cors   
- pip install Flask-SQLAlchemy
- pip install psycopg2 
- pip install fastapi-cors 

Activate environment:
- venv\Scripts\activate


Run API:
- python run.py 

Run Front end:
-  ng serve --open
